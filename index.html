Ejercicio 1: Entendiento CSS

1.1- ¿Qué significa CSS?

CSS HOjas de estilo de Cascada.

1.2- ¿Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS?

El CSS es un lenguaje informatico que se enfoca en el diseño de una webs.

1.3- ¿A qué se llama regla de CSS?

Una regla es la modificacion grafica que el programador quiera aplixar a su codigo.

1.4- ¿Cuál es la anatomía de la regla de CSS? Agregar imagen explicativa y utilizando una lista descriptiva explicar sus elementos.

Selector
Selecciona el o los elementos a dar estilo.
Declaración
Una sola regla especifica a cuál de las propiedades del elemento quieres dar estilo.
Propiedades
Maneras en las cuales puedes dar estilo a un elemento HTML.En CSS, seleccionas qué propiedad quieres afectar en tu regla.
Valor de la propiedad
A la derecha de la propiedad, tienes el valor de la propiedad.

1.5- ¿Qué es un selector en CSS?

Hay muchos tipos de selectores. Un selector es la accion de seleccionar una parte del codigo HTML y aplicarle CSS.

1.6- Identificar y definir los selectores: de ID, de clases, universal, de etiquetas, descendente.

ID
Selecciona el elemento con el ID especifico.
De Clases
Selecciona los elementos que tengan una clase en especifico.
Universal
Nos permite aplicar estilos a todos los elementos del codigo.
De etiquetas
Selecciona una etiqueta y se aplican las propiedades.
Descendente
Seleciona un elemento que está dentro de otro del mismo tipo.

1.7- Explicar brevemente utilizando una lista ordenada: ¿cómo funciona el CSS?

  1. El navegador carga el HTML.
  2. Transforma el HTML en un DOM.
  3. Busca la mayor parte los elementos HTML o vinculados con este.
  4. Analiza el CSS y ordena en diferentes «cubos» las diferentes reglas según el tipo de selector.
  5. El árbol de renderización presenta la estructura en que los nodos deben aparecer después de aplicarle las reglas.
  6. En la pantalla se muestra el aspecto visual de la página.
